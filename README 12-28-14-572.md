# Warbler
(a.k.a. Warbler-Star)

Demo: https://warbler-star.herokuapp.com/

Warbler is a Python-based Twitter clone built using Flask and SQLAlchemy.

### Technologies:
- Flask
- SQLAlchemy
- PostgreSQL
- WTForms
- Bcrypt
- Jinja

### Features:
- Login/Logout
- Liking/Disliking
- Following/Unfollowing
